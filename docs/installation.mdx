# Setup


## Requirements
Before installing, make sure your computer meets the following requirements:

- Python 3.10 or higher


## Install with `pip`

Clone the repository and install the dependencies:

1. Clone the repository using the following command
```bash 
git clone https://github.com/neokd/NeoGPT.git
```

2. Navigate to the root directory of the repository

```bash 
cd NeoGPT
```

3. Install the dependencies

```bash
pip install -e .
```





## Terminal Usage

<Tip>
    If you are running the program first time add `--build` flag to the command. This will build the vector database.
</Tip>

After installion, you can start a chat with the following command:

```bash
python main.py
```


## GUI Usage

To start the GUI, run the following command:

```bash
python main.py --ui
```
